
по умолчанию надо создать 3 файла в текущей директории integers.txt, floats.txt, strings.txt
опции:
    -o задает путь куда положить итоговые файлы
    -p задает префиксы для названий итоговых файлов
    -а вместо создания новых файлов дозаписывает в существующие если такие есть
    -s сбор краткой статистики 
        - количество элементов записанных в каждый файл
    -f сбор полной статистики 
        - для чисел содержит (отдельно для целых и вещественных) минимальное, максимальное, сумму всех чисел и среднее всех чисел
        - для строк размер самой короткой и самой длинной строк


Тесты:
java -jar util.jar

java -jar util.jar -o

java -jar util.jar -p

java -jar util.jar -a

java -jar util.jar -s

java -jar util.jar -f
