Версия jdk 21

Чтобы собрать jar архив для запуска нужно выполнить следующие две команды
javac cft.java
jar cmvf META-INF/MANIFEST.MF util.jar cft.class

После этого можно выполнить программу в консоли передавая желаемые параметры
java -jar util.jar

Примечание:
Создан bat файл для быстрого создания jar архива, запуск .\build.bat
MANIFEST.MF должен оканчиваться пустой строкой
